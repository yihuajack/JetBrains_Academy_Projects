Decrement

Write a sum function that returns the sum of three variable values, where each is decreased by one. Use variables a, b and c as function parameters.

Remember the difference between prefix and postfix increment.

Sample Input:
10 11 -2

Sample Output:
16


Sample Input:
1 1 0

Sample Output:
-1



内存限制: 256 MB
时间限制: 15 秒
